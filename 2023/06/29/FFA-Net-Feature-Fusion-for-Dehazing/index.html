FFA-Net: Feature Fusion Attention Network for Single Image Dehazing

FFA-Net结构由三部分组成：

1.novel Feature Attention 包含 通道注意力with 像素 注意力机制

考虑到 不同通道-wise 特征包含不同的权重信息 and haze分布是不均匀的

这样FA对不同的特征和像素进行不同的处理，为处理不同类型的信息提供了额外的灵活性。

2.对 basic block 结构进行的改进，包括：

-Local Residual Learning
-Feature Attention

LRL 允许不重要的信息 such low-frequency 通过多个残差连接，以此让网络更加focus重要的信息。

3.采用基于注意力的不同层次特征融合结构 FFA 从FA中自适应地学习特征权重，赋予重要特征更多的权重。保留浅层信息并传递到深层。

但雾在图像上分布不均匀，非常薄的雾应该与厚雾区域像素的权重有显著差异。此外，DCP还发现，一些像素在至少一种颜色（RGB）通道中具有非常低的强度，这进一步说明了不同的通道特征具有完全不同的加权信息。

通道的信息差异表现在RGB色彩差异上。

贡献

1.提出端到端特征融合注意力网络FFA-Net。其在纹理细节丰富区域表现尤为突出。在恢复图像细节和颜色保真度方面有优势。


2.提出FA模块，结合了通道注意力和像素注意力。更多地关注了厚雾霾and更重要的通道。


3.提出了一个由局部残差学习和FA组成的基本块ResBlock。=允许通过跳跃连接绕过thin haze region 和 低频信息=


4.FFA结构可以保留浅层信息并将其传递到深层。
